# 🤖 Telegram Trading Analysis Bot

Bot Telegram per analisi tecnica di qualsiasi asset finanziario su Yahoo Finance.

## 📊 Funzionalità
- Analisi tecnica completa di azioni, cripto, forex, ETF, indici
- Supporto per tutti i ticker Yahoo Finance
- Indicatori: RSI, MACD, Medie Mobili, Bande Bollinger
- Segnali di trading automatici
- Ricerca per nome o simbolo
